

# TapAnimator

TapVisualizerの0.2ヴァージョン


## Description
ヴァージョン0.1は階層を深くしてしまったので、まずはシンプルに実装.

v0.2では2Dだけの実装を目指す.

## 実装メモ

 - TapVisualizer__0_2　：削除予定
 - TempChartTest : 削除予定
